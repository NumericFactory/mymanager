## A propos de l'API FactureHero

L'API FactureHero a pour but de faciliter et permettre la construction d'application front-end (web ou mobile) sans se soucier des exigences serveurs, sécurité et base de données. Ainsi depuis votre application front-end vous pouvez appeler n'importe quelles de ces méthodes :

## Customers
- [get] /customers
- [get] /customers/[id]
- [post] /customers/contacts
- [post] /customers/contact
- [put] /customers/contact
